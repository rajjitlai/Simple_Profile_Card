# Simple Profile Card Template

A modern, glassmorphism-styled profile card template with a 3D tilt effect.

## Features
- **Glassmorphism Design:** Translucent card over a background image.
- **3D Interactive Tilt:** Uses VanillaTilt for smooth hover effects.
- **Responsive:** Centered layout using Flexbox.

## How to Customize
1. **Profile Picture:** Replace the `<i class="fa fa-user-circle profile-icon" aria-hidden="true"></i>` in `index.html` with your own `<img>` tag or another icon.
2. **Personal Info:** Update the `<h2>` and `<p>` tags with your name and description.
3. **Links:** Update the `href` attributes in the social links and the main button.
4. **Background:** Replace the `img/bg.png` file or update the `background-image` in `css/style.css`.
5. **Template Ribbon:** Remove the `<div class="ribbon">...</div>` from `index.html` when you are done.

## Optimization
- Google Analytics has been removed.
- Dummy data is used by default.
